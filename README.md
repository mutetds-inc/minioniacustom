## A Minecraft Mod for MINIONIA SEASON 5 / Майнкрафт Мод для MINIONIA SEASON 5

Данный мод содержит все ресурсы которые ***предлагали игроки!***

> - **Новые предметы**
> - **Новые пластинки**
> - **Новые руды**
> - **Новые оружия и инструменты**
> - **И много ещё чего!**
